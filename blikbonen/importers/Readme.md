# Beancount Importers

This package contains importers for a number of financial services. Check the python documentation for an example configuration of each.

Contents:

| Name | Package | Purpose |
|------|------|-------------|
| ASN Bank Chequing | `importers.asnbank.chequing` | ASN Bank Account `.csv` files. |
| ASN Bank Investing | `importers.asnbank.investing` | ASN Bank Investor Account `.csv` files. |
| Coinbase | `importers.coinbase` | Coinbase Statements `.csv` |
| DeGiro Account | `importers.degiro.account` | DeGiro Account Overview `.csv` files. |
| DeGiro Portfolio | `importers.degiro.portfolio` | DeGiro Portfolio (for balance entries). |
| NS Invoice | `importers.ns.invoice` | Nederlandse Spoorwegen Invoice `.pdf` files. |
| NS Travel History | `importers.ns.travel_history` | Nederlandse Spoorwegen `.csv` travel history. |
| Revolut | `importers.revolut` | Revolut Transactions `.csv` |
